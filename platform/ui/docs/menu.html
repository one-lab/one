Sinosoft UI Menu Plugin

Menu - 控件说明

Menu插件能让您只需要写几个html标签即可制作出功能强大，界面美观，支持IE、firefox、ChromeDev等常用浏览器的菜单。它可以读取xml格式、json格式、html格式的数据来生成菜单，而且还可给某个菜单绑定您所要处理的事件。

Menu - 参数配置
参数名 描述 默认值 可选址
dataType 数据类型 "JSON" JSON | XML | HTML
url 请求路径 ""
type 请求方式 "GET" GET | POST
loadText 加载提示 "加载中，请稍后..."
width 弹出的菜单的宽度 110 int类型的数据
eventType 菜单弹出所需要的事件 "click" clikc | mousemove
animation "toggle" toggle | slideToggle | fadeToggle
