Sinosoft UI Menu Plugin

Menu - 控件说明

Menu控件，简单易用，只需指定数据类型和数据URL即可自动生成菜单。支持读取json、xml、html数据格式。灵活的自定义事件扩展，满足复杂的业务需求，支持整体绑定、某一节点单独绑定、也可以通过数据传输绑定事件。

Menu - 参数配置
参数名 描述 默认值 可选址
dataType 数据类型 "json" json | xml | html
url 请求路径 "" string
type 请求方式 "get" get | post
loadText loading加载提示内容 "加载中，请稍后..." string
width 子菜单宽度 110 number
eventType 触发事件类型 "click" clikc | mousemove
animation 菜单展开动画类型 "toggle" toggle | slideToggle | fadeToggle
