Tips - 控件说明

Tips控件方便您为链接或其它元素添加提示标签功能，它可以附在任意元素（标签）之上，并定制展示方式与位置,支持多种浏览器，可定制。功能分为三种：toolTip、customTip、windowTip

  • toolTip：当元素包括title或alt属性时，它的内容将自动变成tip的内容。
  • customTip：用户可自定义Tips显示的内容，支持HTML，这使得其易于定制。触发获取，便于更新动态内容。
  • windowTip：主要用于系统反馈信息提示，支持HTML，可以指定tips与触发元素之前的距离、汽泡自动消失时间，tips窗宽度等。

Tips - 参数配置
参数名 描述 默认值 可选址
type 显示类型 tooltip tooltip | windowTip | customTip
content 显示内容 “” auto | html
title 标题 "系统提示" string
width 固定宽度 "" number
maxwidth 最大显示宽度 "" number
maxHeight 最大显示高度 "" number
pointer 是否显示指针 true true | false
tipPostion 默认显示位置 top top | right | bottom | eft
winTipSite winTip默认显示位置 center leftTop | leftBottom | rightTop | rightBottom | center
hideTime winTip显示多久隐藏(毫秒) 1000 number
autoHide 自动隐藏 true true | false
closeBtn 显示关闭按钮 true true | false
