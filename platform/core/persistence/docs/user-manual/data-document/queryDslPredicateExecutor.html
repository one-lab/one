QueryDslPredicateExecutor

JPA2提供了一个标准的API，可用于以编程的方式构建查询

编写一个criteria实际上是为领域类定义了一个where查询

按照规范，只需要让自定义的仓库接口继承QueryDslPredicateExecutor接口即可

QueryDslPredicateExecutor接口允许使用多种方法来执行规范

例如，findAll()方法可以返回所有符合规范的实体：Iterable<T> findAll(Predicate predicate);

使用规范可以很容易地针对实体建立一套可扩展的查询
