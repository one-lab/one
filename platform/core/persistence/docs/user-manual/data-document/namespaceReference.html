<repositories>

<repositories>元素可以作为<repository>元素的容器，也可以触发仓库实例的自动查找

<repositories>元素内定义的属性可以作用于<repository>元素内，也可以被<repository>元素内定义的属性所覆盖

<repositories>元素有以下几个属性：

  • base-package：定义继承*Repository的仓库接口所属的基础包，可自动查找基础包及其子包
  • repository-impl-postfix：定义自定义仓库接口的实现类的后缀，默认是Impl
  • query-lookup-strategy：定义查询策略，默认是create-if-not-found
<repository>

<repository>元素包括<repositories>元素的所有属性（base-package除外）

所以在<repository>元素内定义的属性值会覆盖在<repositories>元素内定义的属性值

<repository>元素有以下两个扩展属性：

  • id：定义Bean的id
  • custom-impl-ref：定义一个Bean（自定义仓库接口的实现类）的引用
