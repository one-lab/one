对HTML表单的要求
  • 在页面上，上传表单要写上enctype="multipart/form-data",method="POST" 两个属性。
  • 要有type为file的input用于选择文件，file类型的input个数不限
单个文件的上传

单个文件的上传只要form中有file类型的input，并且不同的input name不同

注意form必须添加enctype="multipart/form-data"
